## 1.0.0

### Features
- **DocxView Widget**: Main viewer widget with zoom, scroll, and search support
- **DocxViewWithSearch**: Pre-built widget with integrated search bar
- **DocxSearchController**: Programmatic search control
- **Widget Builders**: 
  - ParagraphBuilder: Full text formatting, hyperlinks, highlights
  - TableBuilder: Tables with borders and cell shading
  - ListBuilder: Bullet/numbered lists with 9-level nesting
  - ImageBuilder: Block and inline images
  - ShapeBuilder: 70+ preset shapes with CustomPaint
- **EmbeddedFontLoader**: OOXML font deobfuscation support
- **DocxViewTheme**: Light/dark presets, full customization
- **DocxViewConfig**: Zoom, search, selection, font fallbacks
